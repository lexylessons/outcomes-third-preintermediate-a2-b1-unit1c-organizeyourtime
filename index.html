Zero conditionals

Use
We use zero conditionals to talk about general truths (things which are always true) or when one thing automatically follows another. A zero conditional has two parts: the if clause and the result clause.

Form
A zero conditional is if + present simple + present simple.
If you want to start a business, you need a clear business plan.
If the printer ink is low, the red light comes on.

We can also put the if clause after the main clause. In this case, don't use a comma.
I always go to bed early if I have an early start the next day.
You don't pass the exam if you don't get 50%.

Note that we sometimes use can('t) in zero conditional sentences.
If I drink too much coffee during the day, I can't sleep.
